# CSS Advanced - SmileSchool

## Description

This project builds on the **HTML Advanced** project by adding full CSS styling to the SmileSchool webpage. The goal is to implement a pixel-perfect design from a Figma designer file using **pure CSS only** — no frameworks, no libraries, just clean semantic HTML and CSS.

## Live Demo

Deployed via GitHub Pages: [View SmileSchool Live](https://enzoasaph.github.io/alu-web-development/css_advanced/)

## What This Project Covers

- How to add style to elements using CSS selectors
- Understanding CSS classes and how to apply them
- Computing CSS Specificity Values
- Box model properties (margin, padding, border, width, height)
- CSS Flexbox for layout
- How browsers load and apply stylesheets
- Responsive design principles

## Page Sections & Styling

| Section    | Key Style Features                                          |
|------------|-------------------------------------------------------------|
| Header     | Transparent, flexbox, white nav links, positioned over hero |
| Banner     | Full background image, large h1, purple CTA button          |
| Quote      | Purple background, circular author image, white text        |
| Videos     | White background, card layout, star ratings, purple accents |
| Membership | Dark background, purple icons, centered items, purple button |
| FAQ        | White background, 2-column grid layout                      |
| Footer     | Dark background, logo + social links, small copyright       |
