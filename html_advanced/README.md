# HTML Advanced

## Description

This project focuses on building a fully structured webpage from a designer wireframe using **pure HTML only** — no CSS, no JavaScript, no external libraries.

The goal is to implement semantic HTML structure that mirrors a real-world design mockup provided through Figma. This trains the skill of reading a design file and translating it into proper, clean HTML markup.

## Wireframe Preview

![School Page Wireframe](https://www.holbertonschool.com/holberton-logo.png)

## What This Project Covers

- How to create an HTML page from a wireframe/designer file
- The purpose of semantic HTML tags (`header`, `main`, `section`, `footer`, `article`, `blockquote`, etc.)
- Proper use of heading hierarchy (`h1`, `h2`, `h3`)
- Understanding the DOM and how browsers parse HTML
- W3C-compliant markup

## Page Sections

| Section     | Description                                       |
|-------------|---------------------------------------------------|
| Header      | Logo and navigation links                         |
| Banner      | Hero section with heading, subtitle, and CTA      |
| Quote       | Testimonial with author image and blockquote      |
| Videos      | Most popular tutorials with ratings               |
| Membership  | Free membership benefits section                  |
| FAQ         | Frequently asked questions in a 2x2 grid          |
| Footer      | Logo, social links, and copyright                 |

## Technologies Used

- HTML5 only (no CSS, no JavaScript)
